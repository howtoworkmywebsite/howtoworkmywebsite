üîó What is a URL?

URL stands for "Uniform Resource Locator" - it's the address of a website.

Example: https://www.example.com/page

  • https:// - The protocol (how to connect)
  • www.example.com - The domain name (which computer to connect to)
  • /page - The specific page you want

üåê How Browsers Work

When you type a URL in your browser:

  1. Your browser sends a request to a server
  2. The server sends back HTML, CSS, and JavaScript files
  3. Your browser reads and displays these files
  4. JavaScript makes the page interactive

üíª Client vs Server

Client (Your Browser):

  • Displays the website
  • Runs JavaScript
  • Handles user interactions

Server (Remote Computer):

  • Stores website files
  • Processes requests
  • Sends data to browsers
